# Lassi & Latte Lounge Website

A simple multi-page café website built using pure HTML. This project represents a fictional café called "Lassi & Latte Lounge", combining traditional lassi drinks with modern coffee culture.

---

## Live Preview
https://github.com/yo-meharry/html_first_project_without_css/blob/main/index.html

---

## Features

- Home Page
  - Introduction to the café
  - About section
  - Menu table with pricing
  - Embedded videos

- Store Hours Page
  - Weekly schedule
  - Simple layout

- Contact Page
  - Contact form (Name, Email, Message)
  - Location and phone details

---
